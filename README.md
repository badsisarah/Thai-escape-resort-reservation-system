# 🌴 Thai Escape Resort Reservation System

## 📌 Description
Application web de réservation de resort en Thaïlande incluant la gestion des chambres, des activités, des paiements simulés et un tableau de bord administrateur.

---

## 🎯 Objectifs du projet
- Développer une application web complète de réservation
- Concevoir une interface utilisateur dynamique
- Gérer les réservations de chambres
- Gérer les activités associées aux séjours
- Implémenter un système de paiement simulé
- Créer un espace client et un espace administrateur

---

## ⚙️ Fonctionnalités

### 👤 Côté client
- Consultation des destinations
- Affichage des chambres disponibles
- Réservation de chambres
- Demande d’activités
- Suivi des demandes
- Paiement simulé
- Historique des paiements
- Espace client

### 🛠️ Côté administrateur
- Gestion des réservations
- Validation des réservations
- Gestion des arrhes
- Validation des activités
- Suivi des demandes
- Gestion des paiements

---

## 📂 Structure du projet

- /data → fichiers JSON (réservations, paiements, activités...)  
- /assets → images et ressources  
- *.php → logique backend  
- *.js → scripts frontend  
- *.css → styles  

## 💳 Paiement
Le système de paiement est une **simulation**.  
Aucune transaction réelle n’est effectuée.  
Les paiements sont enregistrés dans des fichiers JSON.
